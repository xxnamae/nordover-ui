Web Styleguide

Rammeverk for marketing-nettsider. Skandinavisk minimalisme, dramatisk fluid typografi (opp til 160px), flate knapper, generøs editorial luft. @layer-arkitektur, OKLCH gråskala, motion-system, universell a11y. Light default — toggle oppe til høyre.
