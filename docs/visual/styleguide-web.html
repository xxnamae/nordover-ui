FAQ

Kan jeg bruke begge pakker?

Ja, velg web eller app basert på bruksområde. Begge har identisk arkitektur.

Hvordan override farger?

Opprett en brand.css fil og override --color-* tokens i @layer brand blokk.

Støttes dark mode?

Ja, via CSS :has() selector. Toggle med checkbox id="dark".
