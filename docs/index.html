✨ Features

  • ✓ 30+ building blocks (buttons, forms, tables, modals, cards, layouts, etc.)
  • ✓ Semantic tokens (colors, typography, spacing, motion, shadows)
  • ✓ Dual packages (web + app with design-appropriate defaults)
  • ✓ WCAG AA accessible (keyboard navigation, focus management, screen readers)
  • ✓ Dark mode support (via CSS :has() selector, tonal elevation)
  • ✓ Responsive design (mobile-first, touch targets 44px+, container queries)
  • ✓ Motion accessible (respects prefers-reduced-motion)
  • ✓ Production-ready (~104KB combined, optimized CSS)
