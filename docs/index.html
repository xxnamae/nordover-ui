✨ Features

  • ✓ 40+ components (buttons, forms, tables, modals, cards, etc.)
  • ✓ Dual packages (web + app with design-appropriate defaults)
  • ✓ WCAG AA accessible (keyboard navigation, focus management, screen readers)
  • ✓ Dark mode support (via CSS :has() selector)
  • ✓ Responsive design (mobile-first, touch targets 44px+)
  • ✓ Motion accessible (respects prefers-reduced-motion)
  • ✓ Semantic tokens (colors, spacing, typography, motion)
  • ✓ Production-ready (~50KB total, 380 selectors)
