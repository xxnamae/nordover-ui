2. Token System 10 min

Alle farger, spacing og typografi styres av CSS-variabler:

button { background: var(--color-accent); padding: var(--space-4) var(--space-5); border-radius: var(--radius-md); }

Token-kategorier:

Farger

  • --color-bg
  • --color-accent
  • --color-muted
  • --error, --success

Spacing

  • --space-1 til --space-48
  • --gap-tight (8px)
  • --gap-component (24px)

Typografi

  • --text-xs til --text-8xl
  • --fw-display til --fw-caption
  • --font-sans, --font-display
Pro: Endre en token for å påvirke hele nettsiden.

3. CSS Klasse-konvensjoner 5 min

Mønster: komponent-variant-størrelse

.btn /* base */ .btn-primary /* variant */ .btn-lg /* størrelse */ class="btn btn-primary btn-lg"

Komponenter

.btn, .field, .form-input

Varianter

.btn-primary, .btn-secondary, .btn-ghost

Modifikatorer

.btn-lg, .btn-sm, .stack, .cluster

4. Dark Mode 3 min

Dark mode bruker `:root:has(#dark:checked)`:

<input type="checkbox" id="dark"> <label for="dark">🌙 Dark</label>
:root { --color-bg: #FFF; --color-fg: #1a1a1a; } :root:has(#dark:checked) { --color-bg: #0a0a0a; --color-fg: #f5f5f5; }
Viktig: Checkboxen må ha `id="dark"` og være i DOM.

5. Responsive Design 5 min

Mobile-first. Breakpoints:

StørrelseBreakpoint
Mobile0–36rem
Tablet36rem–48rem
Desktop48rem–60rem
Wide60rem+

Eksempel:

.hero { padding: var(--space-4); } @media (min-width: 48rem) { .hero { padding: var(--space-8); } }

Touch-mål: 44×44px minimum.

6. Accessibility WCAG AA 5 min

Keyboard

Tab, Enter, Escape fungerer automatisk.

Dark Mode

Respekterer prefers-color-scheme.

Fokus

2px blå fokus-ring på alle elementer.

Test:

  • Tabb gjennom alle knapper
  • Slå på/av dark mode
  • Test fargekontrast med axe
  • Kjør screenreader

7. Troubleshooting 5 min

Farger endres ikke i dark mode

  • Sjekk at <input id="dark"> er i DOM
  • Bruk var(--color-*), ikke hardkodet farger

Spacing ser rart ut

  • Bruk gap: for flexbox
  • Bruk .stack/.cluster primitiver

Komponent ødelagt

  • Verifiser CSS-filer er linket
  • Inspiser i DevTools

7. Troubleshooting 5 min

Colors don't change in dark mode

  • Check that <input id="dark"> is in the DOM
  • Use var(--color-*), not hardcoded colors

Spacing looks weird

  • Use gap: for flexbox
  • Use .stack/.cluster primitives

Component looks broken

  • Verify CSS files are linked
  • Inspect in DevTools
