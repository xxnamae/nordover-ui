Hva er inkludert

Fluid typografi

CSS clamp() skaler fra mobil til desktop uten breakpoints. xs til 8xl, variable fontvekter.

Light & dark mode

Én mekanisme via CSS :has() og ett checkbox. localStorage-persistens og system-preferanse.

Accessibility first

WCAG AA-kontrast. Fokus-ringer, reduced-motion og keyboard-navigasjon innebygd.
