Hva er inkludert

Fluid typografi

CSS clamp() skaler fra mobil til desktop. 9 trinn fra xs til 8xl, variable fontvekter.

Light & Dark mode

Tema-toggle via CSS :has() og ett checkbox. localStorage-persistens og system-preferanse.

Accessibility first

WCAG AA-kontrastforhold. Fokus-ringer, reduced-motion og keyboard-navigasjon innebygd.

Section patterns

Hero, Feature Grid, CTA, Pricing, FAQ, Service List, Footer — klare til bruk.
