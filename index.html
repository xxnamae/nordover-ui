Design System Features

Accessibility First

WCAG AA-kompatibel. Fokus-indikatorer, contrast-ratio validering og reduced-motion støtte innebygd.

Fluid Typography

CSS clamp() skaler typografi flytende fra mobil til desktop uten breakpoints.

CSS Custom Properties

Alle tokens som CSS-variabler. Enkel theme-override uten å rebygge.

Light & Dark Mode

Innebygd tema-toggle via CSS :has() og localStorage-persistens.
