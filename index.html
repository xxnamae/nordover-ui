Hvorfor Nordover

Tilgjengelig først

WCAG AA. Fokus-indikatorer, kontrast-validering og reduced-motion innebygd.

Fluid typografi

CSS clamp() skalerer typografi flytende fra mobil til desktop uten breakpoints.

CSS-variabler

Alle tokens som CSS custom properties. Enkel theme-override uten å rebygge.

Lys & mørk modus

Innebygd tema-toggle via CSS :has() — ingen JavaScript nødvendig.
