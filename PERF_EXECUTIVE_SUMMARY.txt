================================================================================
WHAT COULD BE BETTER (Low Priority)
================================================================================

⚠️  Color transitions: Some use color-mix() (slight repaint cost)
    → Solution: Pre-compute color tokens instead of animating color-mix()

⚠️  Dark mode backdrop: Uses display:none (reflow trigger)
    → Solution: Use opacity + pointer-events instead

⚠️  Font loading: No explicit font-display:swap (if externally hosted)
    → Solution: Add font-display:swap to @font-face

⚠️  Unused tokens: 86 tokens not currently used
    → These are intentional (reserved for theme customization)
